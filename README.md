# Date Difference Calculator

A simple, beautiful web app to calculate the difference in days between two ISO dates.

## Features

- Calculate days between two ISO date formats
- Supports both date-only (`YYYY-MM-DD`) and full ISO datetime (`2025-12-09T15:35:22.225Z`) formats
- Reduce day difference by 75% and get the resulting date
- Click to copy both the day count and resulting date
- Modern, responsive design with custom color palette

## Usage

1. Enter your start date in ISO format
2. Enter your end date in ISO format
3. Click "Calculate Days" to see the difference
4. Optionally click "Reduce by 75%" to calculate 25% of the difference
5. Click any result to copy it to your clipboard
